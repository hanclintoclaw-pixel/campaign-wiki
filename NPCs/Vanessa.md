# Vanessa

## Overview

A developing smart-agent/vehicle-support project Mevin began building as an attempt to create a van-based agent approaching a self-piloting helper system.

## Known Facts

- Vanessa was envisioned as a smart Agent that would live inside a van.
- Mevin's stated dream was to make the vehicle feel close to a self-piloting "Knight Rider" style platform.
- A full semi-autonomous knowbot vehicle was understood to be beyond what a player character could reasonably create at this stage.
- Mevin therefore began with a more achievable agent-based architecture.
- The surviving design notes preserve an initial mechanical build sketch:
  - Mevin Computer (Programming): 6
  - Agent Frame Core Rating: 6
  - Persona Points: 12
  - Frame Points: 36
  - Pilot Rating target: 6
