## Notes

- Keep `visibility` honest: use `player-safe` for public-facing pages, `true-names` for out-of-character / rules-facing layers, and `gm-facing` only when the page should not be treated as in-character knowledge.
- Prefer short, source-backed facts over long prose dumps.
- Mark uncertainty with `canon_status`, `confidence`, and explicit open questions.
