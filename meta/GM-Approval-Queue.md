# GM Approval Queue

Use this page for wiki or tool changes that are mostly ready but need explicit GM confirmation before they become canon, get deployed, or mutate another tracker.

Do **not** leave approval-waiting edits as uncommitted mystery diffs. If a change is waiting on the GM, either commit it here as a queue item or keep it out of the campaign-wiki worktree.

## How to use

1. Add a row when a change needs GM approval before canonization or deployment.
2. Keep the row short and link the affected page/tool.
3. Use `Pending GM`, `Approved / committed`, `Rejected`, or `Superseded` for status.
4. Once approved, commit the real page changes and update the row with the commit or outcome.
5. If the item is GM-only or spoiler-sensitive, do not place the details here; add a player-safe placeholder and keep the private note outside the public wiki.
