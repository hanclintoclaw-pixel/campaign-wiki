## Host feel

The host feels busier than it should for a corner store. It is bright, cute, overdecorated, and constantly trying to sell the user something: lucky-cat mascots, snack banners, collectible-card animations, tiny coupons, and popup chatter from nearby public users.

Under that cheerful clutter, the host is still a real neighborhood information knot. A patient decker can use the noise as cover, but the same noise makes it hard to separate actual **Pixel Sticks** signal from teenage posturing and ordinary retail spam.
